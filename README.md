# MrDickButt

Custom Codex pet package for MrDickButt.

## Reference Material

MrDickButt is inspired by Dick Butt from KC Green's original comic. All credit for Dick Butt and the source comic goes to KC Green.

## Install

Copy the contents of `pet/` into a custom Codex pet folder named `mrdickbutt`, then reload Codex and select `MrDickButt`.
